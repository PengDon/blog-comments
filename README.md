[博客框架](https://hexo.io/zh-cn/)

- npm install hexo-cli -g
- hexo init blog
- cd blog
- npm install
- hexo server

# 写作
hexo cl && hexo g && hexo s
# 部署
hexo cl && hexo g && hexo d


## 说明 

 参数 | 功能 | 路径
 ---- |----- |----
 post |	新建文章 |	/source/_posts/
 draft |	新建草稿 |	/source/_drafts/
 page |	新建页面（标签页，分类页等） |	/source/

* 新建文章
hexo new post 文章名称
